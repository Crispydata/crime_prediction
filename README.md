# crime_prediction

This project was created to predict the number of potential offenders in 2026 in Victorian suburbs using the past data from https://www.crimestatistics.vic.gov.au/.

Source of data used for this project:
1. Victoria crime data
Crime Statistics Agency Data Tables - Unique Alleged Offenders - Unique Alleged Offenders - year ending June 2025 - Data.gov.au
https://data.gov.au/data/en/dataset/agency-data-tables-unique-alleged-offenders/resource/f836a654-d753-4fdd-a617-119ffed47f05

2. ABS data for victorian suburbs
https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/allocation-files

Tools used:
1. VS Code
2. Git Hub Desktop
3. Power BI Desktop

Languages used:
1. Python

Models used:
1. RandomForestRegressor

Features:
1. Country of Birth
2. Age group

Conclusion:
Due to insufficient features in the crime data, predicted values were too high in new data.
